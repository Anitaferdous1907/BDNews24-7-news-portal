Difference between var let and const

The following are the distinctions between var, let, and const variable declarations in JavaScript: Variables declared with var and const are only available to the current function body. Hoisted variables are those declared with the var keyword. Hoisting indicates that the variable can be accessed in its enclosing scope before it is declared. Variables declared with the let keyword are block-scoped, which means they have scope only to the block in which they are declared.
